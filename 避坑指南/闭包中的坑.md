1.闭包中的传入外部参数不可修改
看一下实例

```
@test function testAB()
    var a = 1
    let list=new HashList<int>()
    list.add(1,2,3,4)
    list.forEach() (integer t) ->
        a++//外部传入的变量不可修改
    print(a)
```

在闭包内部，局部变量a不能改变，那怎么办呢？
法1：用全局变量
法2：用静态/实例 参数
法3：用reference包

```
@test function a()
    let someList=new HashList<int>()
    let dist_ref = new Reference(0.)
    someList.forEach() (integer t) ->
        if dist_ref.val < 500  
            dist_ref.val += t
```

新建一个Reference对象传入任何类型的值，
但是要注意用完后销毁对象实例！
